步骤 2 / 7

环境初始化向导

自动执行依赖安装与工具准备。Linux 会顺序安装相关组件，Windows 会调用 Easy Install 脚本。

执行内容

  1. 安装 huggingface_hub 与 modelscope
  2. 配置 Node.js 20 源并安装 nodejs
  3. 克隆或更新 ai-toolkit，安装 requirements
  4. 启动 ai-toolkit，Linux 默认端口 6006，Windows 默认端口 7867
Autodl 学术镜像加速

仅适用于 AutoDL 环境，可加速 GitHub 与 HuggingFace 访问；不需要时建议关闭。

GitHub 加速

第三方 GitHub 代理加速，适合访问 GitHub 较慢时使用。
